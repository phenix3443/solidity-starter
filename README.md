solidity contract template repository, configured as [use solidity to develop smart contract](https://blog.panghuli.cn/posts/ethereum/solidity/).

## prepare

you should install following tools:

- pnpm
- pip3
- foundry

```shell
pnpm init
pnpm install:slither
```
